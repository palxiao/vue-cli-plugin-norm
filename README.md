# vue-cli-plugin-norm
Code style and validation, git commit specification. Fully automated!

Currently rely on Vue-cli
```
npm install -g @vue/cli
# OR
yarn global add @vue/cli
```

At your project: 
```
vue add vue-cli-plugin-norm
```

start, enjoy code!

new commit: 
```
npm run c
``` OR 
```
npm run commit
```

commit log view: ```npm run clog```
